# **App Name**: Royaltech Limited

## Core Features:

- Stock Management: Manage laptop stock, including adding new laptops, updating existing ones, and tracking inventory levels.
- Customer Management: Create and manage customer records, including contact information, purchase history, and lease agreements.
- Document Generation: Generate receipts, invoices, proforma reports, and repair notes in a standardized format.
- Lease Tracking: Track laptops leased to customers, including lease duration, payment schedules, and return dates.
- Smart Notifications: Provide proactive notifications based on the status of laptop leases, stock levels, and repair turnaround times, making use of a tool to decide the proper content of the notification.
- Laptop Tracking: Track the laptop's location using Google Maps integration.

## Style Guidelines:

- Primary color: Light grey (#F5F5F5) for a clean, professional look.
- Secondary color: Dark blue (#2c3e50) for headings and important text.
- Accent: Teal (#1abc9c) for highlights, calls to action, and progress indicators.
- Use a clear and professional sans-serif font for all text.
- Responsive layout that adapts to different screen sizes, ensuring usability on desktops, tablets, and mobile devices.
- Use a consistent set of icons throughout the application to represent different actions and categories.
- Subtle transitions and animations to enhance the user experience without being distracting.
